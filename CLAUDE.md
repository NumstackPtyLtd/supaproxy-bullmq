# @supaproxy/bullmq

BullMQ queue adapter for SupaProxy. Implements the QueueService port from @supaproxy/core.

See the [central hub](https://github.com/NumstackPtyLtd/supaproxy) for cross-repo governance.

## Structure
- `src/BullMqService.ts` — QueueService implementation
- `src/index.ts` — createBullMqQueue() factory

## Rules
- All changes go through PRs. Never push directly to main.
